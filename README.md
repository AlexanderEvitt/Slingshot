# Slingshot
## Orbital Dynamics Solver

Slingshot aims to provide a user-friendly astrodynamics experience to those looking to design spacecraft trajectories. Currently in the early phase with basic features.

![Screenshot](screenshot1.png)

## Current Features

- Basic Runge-Kutta4 trajectory propagation in the Earth-Moon system with _N_-body effects
- Basic simulation controls, including time interval and integration length
- Maneuver nodes that allow instantaneous changes in velocity
- A slick UI

## Planned Features

- Multiple planets, all influencing the trajectory
- Plots to allow design of trajectory
- Reference frames
